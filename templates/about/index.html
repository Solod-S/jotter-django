Jotter

A quiet place for notes.

Jotter is a focused Django notes app for capturing thoughts, drafts, and small ideas without extra interface noise.

What it is

The app keeps the basics close: create a note, read it later, edit what changed, and move on. The interface stays simple so the note remains the main thing on the screen.

Design principles

Fast capture

Opening a blank note should feel immediate, with no setup before writing.

Readable by default

Clear type, calm spacing, and strong contrast make notes easy to scan and revisit.

Minimal chrome

Navigation and controls are present when needed, then get out of the way.

How it fits

Use it for short captures, study notes, planning fragments, or longer drafts. Jotter is intentionally text-first and avoids features that slow down writing.
